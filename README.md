# AGENDA ELECTRONICA

## Descripción

Agenda electrónica  con interfaz grafica que permite guardar hasta 10 contactos.

cuenta con 6 campos para rellenar y 3 botones (guardar+ avanzar+retroceder)

## Datos que guarda
1.  DNI
2.  Nombre
3.  Apellidos
4.  Direccion
5.  Telefono
6.  Fecha de nacimiento
